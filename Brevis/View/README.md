# KeyDeck — Xcode Hotkeys Companion

KeyDeck is a small SwiftUI app that helps you learn and memorize Xcode keyboard shortcuts by presenting flashcards in a clean and interactive interface.

## Project Structure

- `ContentView.swift`: The main SwiftUI view displaying the flashcards.
- `Models.swift`: Contains the data models representing Xcode shortcuts.
- `ViewModels.swift`: Handles the logic for managing flashcard state and progression.
- `Assets.xcassets`: Contains images and other asset files used in the app.
- `Info.plist`: Configuration file for the app's metadata and settings.

## Features

- Swipe through flashcards to learn shortcuts.
- Tap on cards to flip between shortcut keys and their descriptions.
- Progress tracking to monitor which shortcuts have been mastered.
